Sri Lanka Chatbot Embed Example

This page demonstrates how to embed the Sri Lanka Travel Chatbot on any website.

How to Use

To add the chatbot to your website, simply include the script at the end of your HTML body:

<script src="chatbot-embed.js"></script>

Customization

You can customize the chatbot by modifying the configuration variables at the top of the script:

const config = {
    chatbotUrl: "https://srilanka-c0d4bhhfegg3cvge.southeastasia-01.azurewebsites.net/",
    chatIconColor: "#4CAF50",
    chatHeaderColor: "#4CAF50",
    chatTitle: "Travel Sri Lanka Assistant",
    chatIconPosition: "right", // 'right' or 'left'
    chatWidth: "350px",
    chatHeight: "500px"
};
